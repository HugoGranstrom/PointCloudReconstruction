Problem description

  • Given sampled points from surface $\Gamma$, create an approximation of $\Gamma$.

Mesh-based methods

  • Intuitive approach to construct mesh from points.
  • In this project we used mesh-free methods.

Radial basis functions

  • Real valued scalar function.
  • Function of the distance between two points.

Distance matrix

  • Constructing a linear system for our interpolation problem.

Distance matrix

$$ A = \phi \left(\begin{bmatrix} r_{1 → 1} & r_{1 → 2} & r_{1 → 3} \\ r_{2 → 1} & r_{2 → 2} & r_{2 → 3} \\ r_{3 → 1} & r_{3 → 2} & r_{3 → 3} \\ \end{bmatrix} \right) $$

Distance matrix

$$ A = \phi \left(\begin{bmatrix} r_{1 → 1} & r_{1 → 2} & r_{1 → 3} \\ r_{2 → 1} & r_{2 → 2} & \color{green}{r_{2 → 3}} \\ r_{3 → 1} & r_{3 → 2} & r_{3 → 3} \\ \end{bmatrix} \right) $$

RBF interpolation

Given input points $x$ and outputs $y$ we can construct a linear system of equations: $Ac = y$ where A is our distance matrix constructed using our points $x$.

Solving this will give us $c$ which is our scaling coefficients for our RBFs.

Numerical stability

  • Positive definite for numerical stability
  • One such RBF is a gaussian kernel $\phi(x)=e^{-\epsilon r^2}$

Numerical stability

  • Compact support for sparsity
  • One such RBF, Wendland's $C^2$ kernel

$$ \phi(r) = \begin{cases} (1 - \frac{r}{\epsilon})^4(4 \frac{r}{\epsilon} + 1) & r <= \epsilon \ 0 & r > \epsilon \end{cases} $$

RBF Interpolation in $\mathbb{R}^n$

  • Easy since Euclidian norm is defined in any $\mathbb{R}^n$.

Curl-free method

Surface normals and potentials

  • If a surface is a level-set to a function will the normals be perpendicular to the gradient
  • If a scalar-function has a gradient is the function a potential
  • All potentials can be shown to be curl free

Curl-free RBF interpolation

  • Interpolate curl-free vector fields
  • Construct Hessian of a scalar RBF $\mathbf{\Phi}(|| \mathbf{x} - \mathbf{x}{center} ||) = -\nabla \nabla^T \phi(|| \mathbf{x} - \mathbf{x}{center} ||)$

Curl-free Method

  • Derive potential anlytically $s(\mathbf{x}) = \sum_{i=1}^N \nabla^T \phi(|| \mathbf{x} - \mathbf{x}_{i} ||)$

  • Unique to a constant $C$.

  • Compensate by subtracting mean potential in points

Curl-free PU

  • Blend potential function, not vector field
  • Compensate for $C$ in each patch.
